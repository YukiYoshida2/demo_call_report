### 課題間の因果連鎖

```
新規メタCV大量投入（W07〜）
  → DIS CN率4週連続下落（58.8% → 32.0%）
  → 永野・村松のCN率週次急落（-20.5pp、-17.1pp）
  → DIS SAL実績 68件(前月) → 45件(当月) ▲33.8%
  → SAL全体でDIS寄与が減少、TOPが全体達成を単独で支える構図に

FAX・EDM送信量の減少 + 不動産CVのSAL率崩壊（3.1%）
  → 着電数-49.4%（540 → 273件）
  → 商談実施着予60%のリスク残留

中本 陽介のTOP SAL低迷（25.0% vs 平均41.3%）
  → 期待SAL比-4.7件の損失
  → TOPが全体を支える中でのエース不調が目標達成余力を削ぐ
```

## 5. ネクストアクション案

### 優先度1（今週中）

**1. DIS新規メタCV群の配信停止または見直し**
- 対象: `democall_meta_entertainment-profile`（113件, CN26.5%, SAL0%）、`democall_meta_business-robot`（16件, CN31.2%, SAL0%）、`democall_meta_entertainment-anime`（7件, CN28.6%, SAL0%）
- アクション: 上記3CV（計136件）の配信を即時停止し、DIS全体CN率への改善効果を確認する
- 期待効果: DIS CN率 44.5% → 推計47〜49%への回復（主力「デモ電話_Facebook・AI」CN44.8%水準に正規化）、DIS SAL達成率 80% → 85〜90%への改善

**2. 中本 陽介のTOP商談設定プロセスのSV実施**
- 対象: 中本 陽介 × TOPチャネル（SAL率25.0%、-13.5pp、期待差分-4.7件）
- アクション: 当月のTOPリードでSALに至らなかったケースのコール録音を5件以上レビュー。ハイパフォーマー（村松 亜茉音: TOP SAL40.4%）との通話比較分析を実施し、W06のSAL急落（-49.3pp）の原因を特定
- 期待効果: -4.7件のSAL機会損失を3月までに-2件以内に削減（チャネル平均の75%水準到達を目標）

### 優先度2（今週〜来週）

**3. LIS業種特化CVの品質改善（医療・福祉 / 飲食業）**
- 対象: `デモ電話_Lis・医療・福祉`（SAL率0%、前月比-42.9pp📉）、`デモ電話_Lis・飲食業`（前月SAL比-29.2pp📉）
- アクション: 医療福祉CVは即時見直し（LPコンテンツ・ターゲティング条件の確認）。飲食業CVは前月比較で何が変わったか広告担当者と協議。コアCV「デモ電話_Lis・法人全般」のパフォーマンスは健全のためLIS全体停止は不要
- 期待効果: LIS SAL率 16.7% → 20〜22%への回復（前月28%を短期目標）

**4. FAX・EDM不動産CVのSAL率立て直し**
- 対象: `デモ電話_FAX_不動産(売買)`（143件、SAL率3.1%、前月比-8.4pp📉）
- アクション: FAX受信企業プロファイル・送信タイミング（W08集中の影響）・担当者対応を確認。並行して好調な「デモ電話_FAX_病院・クリニック」（SAL16.4%、+8.5pp）のアプローチ手法を不動産担当者に横展開
- 期待効果: 不動産SAL率 3.1% → 8〜10%への改善でFAX・EDMチャネルSAL着予の安定維持

### 優先度3（中期: 3月施策）

**5. DIS営業時間外・休日リードのSAL効率改善**
- DIS全体の65.5%（567件/865件）が営業時間外リードで、SAL率は時間内14.3%に対し10.2%（-4.1pp）。休日DISのSAL率は6.9%（平日13.4%比半減）
- アクション: 営業時間外リードへの翌営業日午前優先コールのルール化、および休日投入するDISクリエイティブ・ターゲットの品質基準引き上げを3月施策として検討

**6. 「その他」チャネルのCV振り分け見直し**
- 「デモ電話_迷惑電話LP」「デモ電話_潜在記事」など認知・潜在系CVはデモ電話チームの管理指標に含める性質のものではなく、「その他」のSAL目標（9件）の設定自体が非現実的
- アクション: これらCVの集計対象除外または目標値の実態合わせを運営管理部門と協議
